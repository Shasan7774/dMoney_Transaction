# dMoney_Transaction_API

## Project Summery:
Automated dmoney_transaction APIs using PostMan and NewMan

## Tools and Technologies Used: 
Postman
Newman
newman-reporter-htmlextra

## Prerequisites:
NodeJS

## How to run:
- ``` git clone ```
- ``` npm i ```
- ``` npm test ```
A report will be generated in the project Reports directory
